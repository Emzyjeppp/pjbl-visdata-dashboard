# Kontribusi untuk Proyek Visualisasi Data Yogyakarta

Terima kasih telah tertarik untuk berkontribusi pada proyek PjBL UAS Visualisasi Data Yogyakarta!

## Panduan Kontribusi

1. **Fork Repositori**: Salin proyek ke akun GitHub Anda.
2. **Buat Branch Fitur**: Gunakan penamaan ekspresif (misal: `feat/optimasi-chart` atau `fix/bug-filter`).
3. **Kepatuhan Kualitas**: Pastikan kode yang Anda buat memenuhi standar linter dan formatter sebelum dikirim:
   ```bash
   npm run lint
   npm run format:check
   ```
4. **Kirim Pull Request (PR)**: Berikan penjelasan singkat terkait perubahan yang Anda lakukan.
